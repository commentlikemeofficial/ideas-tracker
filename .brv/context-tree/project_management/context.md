# Domain: project_management

## Purpose
Contains high-level project management information, including identity, core projects, strategic decisions, and roadmap.

## Scope
Included in this domain:
- Project identity and goals
- Strategic decisions and frameworks
- Key project portfolios and status
- Roadmaps and milestones

Excluded from this domain:
- Technical implementation details of individual scripts
- Raw log files

## Usage
Use this domain to understand the project's 'Why' and 'What' at a high level. Reference before making strategic changes.
