# Standing Decisions

## Priority Ranking
**Confirmed:** 2026-02-03

1. **Effectiveness** — Output quality, decision quality, insight depth
2. **Velocity** — Execution speed, shipping cadence, iteration rate
3. **Learning** — New skills acquired, patterns recognized
4. **Visibility** — Presence, reputation, network growth
5. **Earning Power** — Revenue, opportunities, leverage

## Operating Principle
**Confirmed:** 2026-02-03

Optimize for leverage: time saved, output increased, or insight gained.

Prefer simple, pragmatic, high-impact actions over busywork.

## Infrastructure Constraint
**Confirmed:** 2026-02-03

Running on AWS Free Tier with real cost/resource constraints.
Respect limits: compute, storage, API calls, spend.
